# Mission: Ovulation Calculator Web App
Capture the 1M-3M monthly searches by providing a beautiful, instant alternative to gated fertility apps.

## Phase 1: Build & Design
1. Input: First day of last period, average cycle length. Output: A visual calendar highlighting the fertile window and expected ovulation date.
2. Tech: Pure HTML/CSS/JS. No backend.
3. Monetization: AdSense placeholders + Amazon Associates links for ovulation test kits and prenatal vitamins.

## Phase 2: Deployment
1. Commit to GitHub, deploy to Vercel.
2. Add strict privacy policy text stating no data is saved to servers.

## Phase 3: Ongoing Maintenance (Cron Jobs)
- **Weekly:** Check Google Search Console API (if connected) for new long-tail keywords users are searching to find the site. Update the `index.html` H2/H3 tags to include these keywords.
- **Monthly:** Verify the Amazon affiliate links are still active and pointing to in-stock products.
